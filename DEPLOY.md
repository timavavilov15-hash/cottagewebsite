# CONTENT-GUIDE - Гайд по наполнению контентом

## 📝 Общие принципы

1. **Структура контента** - Следуйте утвержденной архитектуре
2. **SEO оптимизация** - Используйте ключевые слова естественно
3. **Доступность** - Добавляйте alt-тексты, описания
4. **Консистентность** - Соблюдайте стиль и тон компании

## 🏠 Главная страница (index.html)

- Картинка должна быть оптимизирована (WebP + JPEG)

- Ограничиться 3-4 основными услугами
- Использовать иконки для визуализации
- Краткое описание (20-30 слов)

### Projects/Portfolio Section
- Показывать лучшие 6-8 проектов
- Использовать высокачественные изображения
- Добавить очень краткое описание проекта

### CTA Section
- Два четких CTA
- Контрастные цвета кнопок
- Максимум 150 слов текста

## 🎨 Рекомендации по медиа

### Изображения
- **Оптимизация**: WebP + JPEG fallback
- **Размеры**: Desktop (1920x1080), Tablet (1024x576), Mobile (640x360)
- **Качество**: 80-85% для JPEG
- **Alt-текст**: описывает содержимое, содержит ключевые слова

### Видео
- **Формат**: MP4 + WebM
- **Размер файла**: max 5MB для hero видео
- **Thumb**: изображение-плакат (poster)
- **Длительность**: 15-30 сек для hero, 1-2 мин для демонстрации

### Иконки
- **Формат**: SVG (масштабируется без потерь)
- **Размер**: 24x24 - 64x64px
- **Цвет**: использовать currentColor для наследования цвета

## ✍️ Стиль написания

### Тон голоса
- Профессиональный, но дружелюбный
- Избегать жаргона
- Использовать активный залог
- Примеры: "Мы создаём" вместо "Нами создаётся"

### Формат текста
- Короткие предложения (15-20 слов)
- Списки вместо длинных абзацев
- Один основной идея на абзац
- Используйте выделение для ключевых слов
